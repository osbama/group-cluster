=======================
Carbon
=======================


Make & Model 
====================

Carbon is a Supermicro GPU server which has been retired, and serves as the headnode for the cluster. 
It hosts the shared ZFS filesystem. 

Intended use
====================

Carbon handles some of the infrastructure work. It can also be used for light post-processing tasks.
