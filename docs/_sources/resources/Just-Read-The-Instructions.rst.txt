==================================
Just Read The Instructions (JRTI)
==================================


Make & Model 
====================

JRTI is a `Dell R7525  <https://www.dell.com/tr/sletmeler/p/poweredge-r7525/pd>`_. 


Intended use
====================

JRTI is intended to replace Carbon as a GPU server.
