=======================
Triumphant Coal
=======================


Make & Model 
====================

Triumphant Coal is an `ThinkStation P920 Tower <https://www.lenovo.com/tr/tr/workstations/media-and-entertainment-industry/ThinkStation-P920/p/33TS3TPP920>`_. 


Intended use
====================

Triumphant Coal is intended for high-memory long-run jobs, such as GW calculations, Coupled Clusters etc.
